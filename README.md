# Ubuntu Project

This is a public repository running GitHub Actions on Ubuntu with unlimited free minutes.

## Features
- ✅ Running on Ubuntu Latest
- ✅ Public repository (unlimited minutes)
- ✅ GitHub Actions workflows

## Getting Started

Clone the repository:
```bash
git clone https://github.com/thom14413-bit/unbutu.git
cd unbutu
```

## Workflows

Check the `.github/workflows/` directory for available workflows.
